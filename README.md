Welcome.
I have created a mobile application named SparkConnect.

This app allows the user to connect with friends and family by allowing the user to type in the time of day and giving suggestions on what to do.

I developed this app using Android Studio and Kotlin.

As we all know, maintaining a social life can be very difficult, especially for people who have businesses or working individuals.

My SparkConnect app helps bridge this gap by allowing the user to add the time of day(night,afternoon,morning),and it gives you an acitvity or suggestion known as a spark.

This Spark gives the user an action on what to do during that specific time of day.
It makes your life easier.

In order for my app to run seamless and remain consistent I needed to do more research on how to use Github and deploy as well as debug my app.

My project makes use of maintainability, building that is automated and version control which is found in Github.
